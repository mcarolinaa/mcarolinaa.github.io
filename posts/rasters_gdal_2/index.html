Manipulações básicas de arquivos raster usando GDAL e python

O GDAL é uma translator library para dados vetoriais e rasteriais, com modelos próprios para trabalhar esses dados.

Uma grande diversidade de pacotes e libraries em diversas linguagens são baseados no GDAL. Acredito que vale muito a pena saber usá-la até para entender melhor a estrutura dos dados no contexto de GIS.

o GDAL tem API para diversas linguagens, incluindo o python. A API para python pode ser consultada aqui:

https://gdal.org/api/python.html

Nesse post há rotina para as seguintes ações:

  • Reprojetar;

  • Resample;

  • Clipar raster

Resampling é uma operação na qual pegamos os dados (valores) do raster, tiramos do seu grid original, e colocamos em um novo grid (diferentes números de linhas e colunas).

A Reprojeção acaba sendo semelhante, pois conta com um resampling feito em um grid de diferente CRS do raster original.

Já a operação conhecida como clip é apenas cortar o raster usando, em geral, um arquivo vetorial (como shape por exemplo) como referência.

Reprojetar, fazer resample e clipar o raster

from osgeo import gdal
import numpy as np
import matplotlib.pyplot as plt
import glob
import subprocess

Aqui feito de uma vez só pela praticidade.

Abre e faz reprojeção.

ds = gdal.Open('path_to_raster')
array = ds.GetRasterBand(1).ReadAsArray()
plt.imshow(array)
plt.colorbar() # add a color scale
print(ds.GetGeoTransform()) # upper left corner, spatial resolution
print(ds.GetProjection()) # proj system

dsReprj = gdal.Warp('nameRasterOutput.tif',
                    ds,
                    dstSRS = 'EPSG:4326') # epsg p/ o qual quero mudar

Faz resample.

dsRes = gdal.Warp('ndviRes.tif',
                  ds,
                  xRes = 150, # modificando numero de linhas e colunas
                  yRes = 150,
                  resampleAlg = 'bilinear') # Metodo. default: nearest neighbour

E clipa o raster usando um arquivo vetorial (ex: .shp) de referência.

dsClip = gdal.Warp('ndviClip.tif',
                   ds,
                   cutlineDSName = 'path_to_vector_file',
                   cropToCutline = True, # clip to the extent of the vector file
                   dstNodata = np.nan)
array = dsClip.GetRasterBand(1).ReadAsArray()
plt.imshow(array)
plt.colorbar()

Não esquecer de fechar os rasters.

ds = dsClip = dsRes = dsReprj = None
