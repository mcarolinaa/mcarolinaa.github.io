Manipulações básicas de arquivos raster usando GDAL e python

O GDAL é uma translator library para dados vetoriais e rasteriais, com modelos próprios para trabalhar esses dados.

Uma grande diversidade de pacotes e libraries em diversas linguagens são baseados no GDAL. Acredito que vale muito a pena saber usá-la até para entender melhor a estrutura dos dados no contexto de GIS.

o GDAL tem API para diversas linguagens, incluindo o python. A API para python pode ser consultada aqui:

https://gdal.org/api/python.html

Nesse post há rotinas para as seguintes ações:

  • Ler e plotar arquivo raster;

  • Criar binary mask

from osgeo import gdal
import numpy as np
import matplotlib.pyplot as plt
import glob
import subprocess

Ler a plotar arquivo raster

ds = gdal.Open('path_to_raster') # abre somente 1
gt = ds.GetGeoTransform() # object (upper left corner: .. x-coord, horizontal pixel res, y-coord, vertical pixel res)
proj = ds.GetProjection()

# access band to get data
band = ds.GetRasterBand(1) # se for multibanda, especificar a desejada
array = band.ReadAsArray()

plt.figure()
plt.imshow(array)

Criar uma binary mask

Maneira de visualizar/explorar os dados.

binmask = np.where((array >= np.mean(array)),1,0) # tudo que for > média, entao =1
plt.figure()
plt.imshow(binmask)

# ajusta driver p/ exportar
driver = gdal.GetDriverByName('GTiff')
driver.Register()
outds = driver.Create('path_to_binmask.tif',
                      xsize = binmask.shape[1], # number of cols
                      ysize = binmask.shape[0], # number of rows
                      bands = 1,
                      eType = gdal.GDT_Int16)
outds.SetGeoTransform(gt)
outds.SetProjection(proj)
outband = outds.GetRasterBand(1)

# escreve em disco
outband.WriteArray(binmask)
outband.SetNoDataValue(np.nan)
outband.FlushCache()

# não esquecer de fechar o dataset
outband = None
outds = None
