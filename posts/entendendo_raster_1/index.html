Ler um raster

Operações com e entre rasters são bem comuns e importantes.

No python, o uso do Gdal, numpy e pandas já permite a realização de muitas dessas operações.

O código a seguir pega um raster (formato .tif), abre, pega informações sobre coordenadas e resolução e coloca as bandas (considerando a ordem já conhecida) em arrays, para ser usada em histogramas, scatterplots e outros.

import numpy as np
from osgeo import gdal
import matplotlib.pyplot as plt
import seaborn as sn

# Abre
path_raster = 'path/do/raster.tif'
raster = gdal.Open(path_raster)

# Guarda algumas informações importantes
gt = raster.GetGeoTransform()
res = gt[1]
xmin = gt[0]
ymax = gt[3]
xsize = raster.RasterXSize
ysize = raster.RasterYSize
xstart = xmin +res/2
ystart = ymax - res/2

# Pega bandas e coloca em arrays flatten (p/ certas análises e visualizações)
# para ficar com a array multidimensional, excluir o .flatten()
blue = raster.GetRasterBand(1).ReadAsArray().flatten()
green = raster.GetRasterBand(2).ReadAsArray().flatten()
red = raster.GetRasterBand(3).ReadAsArray().flatten()
redg = raster.GetRasterBand(4).ReadAsArray().flatten()
nir = raster.GetRasterBand(5).ReadAsArray().flatten()

# fecha raster
raster = None

# faz histograma de uma banda
plt.figure(figsize=(9,7))
sn.histplot(redg, color='red')
plt.title('Red-Edge Band')
plt.show()
